# Flashcard App

This is a Flashcard Management and Study app designed to help users efficiently create, organize, and study flashcards. The app provides a seamless experience for learning, tracking progress, and staying organized.

# # Flashcard Management

Create Flashcards – Add new flashcards with a question, answer, and category.
Edit Flashcards – Update the details of existing cards.
Delete Flashcards – Remove flashcards that are no longer needed.
Form Validation – Receive helpful validation messages when trying to submit incomplete cards.
View All Cards – Display all flashcards in a clean, responsive grid layout.
Flashcard Details – View detailed information including question, answer, category, and mastery progress.

# # Study Mode

Interactive Learning – Study flashcards one at a time in Study Mode.
Reveal Answers – Click a flashcard to see the answer.
Track Mastery – Mark a flashcard as “I Know This” to track progress on a 0–5 scale.
Navigation – Move between flashcards using Previous and Next buttons and see your current position (e.g., “Card 1 of 40”).
Reset Progress – Restart learning a flashcard by resetting its progress.

# # Filtering & Organization

Filter by Category – Select one or multiple categories to focus your study.
Category Counts – See the number of cards per category in the filter dropdown.
Hide Mastered Cards – Focus only on flashcards that need practice.
Shuffle Cards – Randomize the study order for variety.

# # Statistics & Progress

Study Statistics – View overall progress with total cards, mastered, in progress, and not started counts.
UI & Navigation
Toggle Views – Switch easily between Study Mode and All Cards views.
Load More Cards – Load additional cards when viewing large collections (more than 12).
Toast Notifications – See notifications when cards are created, updated, or deleted.
Responsive Design – Optimized layouts for all screen sizes.
Accessibility – Hover, focus states, and full keyboard navigation for all interactive elements.
This app is perfect for students, learners, and anyone who wants to organize and master information efficiently while keeping track of their learning progress.

# # Tech Used

- React (react, react-dom) – For building the user interface.
- Array Shuffle (array-shuffle) – Utility to shuffle arrays.
